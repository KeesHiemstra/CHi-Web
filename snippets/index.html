Delphi: String library

In allmost all my programs I have to manipulate strings. In the years I have collected the functions that I use most in my ChiStr library. It contains the following functions:

  • BackPos
    Searching backwards
  • LeftStr
    Returns the left of the string from start position
  • RightStr
    Returns the right of the string from start position
  • EndStr
    Returns the remaining string from start position
  • LTrim
    Returns the string without leading spaces and tabs
  • RTrim
    Returns the string without trailing spaces and tabs
  • LongInt2Base36
    Translates an integer to a number based on 36 characters (instead of 2 {binary}, 8 {octal}, 10 {decimal} or 16 {hexadecimal})
  • ReplaceStr
    Replaces all occurences of the given string
  • IntToStrF
    Returns a string from an integer with digitgrouping
  • SplitSeparator
    Splits string on given separator
  • PosAt
    Searching for a string from start position
  • PosNum
    Returns the number of occurences of string
